## What is Tenivel?

Tenivel helps you manage your personal finances — budgets, transactions, savings goals, recurring payments, forecasts — entirely on your own computer. Your data never leaves your device.

Unlike cloud-based finance apps, Tenivel stores all financial data in a local AES-256 encrypted database. An optional account on tenivel.com is used solely for license management and device activation — no financial data is ever transmitted to the server.

## Key Features

- **Bank Import** — CSV and PDF import for major banks (Sparkasse, DKB, ING, Commerzbank, N26, Volksbank) plus a generic fallback
- **Dashboard** — KPIs, spending trends, monthly comparison, year overview, and anomaly detection
- **Monthly Budget Planner** — Set budgets, track fixed costs, category breakdowns with progress bars
- **Savings & Spending Goals** — Milestones, pace indicator, auto-assignment, archive
- **Recurring Payments** — Auto-detection, 30/90-day forecasts, inline editing, pause/resume
- **What-If Simulator** — 8 scenario templates (salary raise, moving, car purchase, job loss, and more)
- **Local AI Analysis** — Optional AI assistant via local Ollama instance (your data stays on your device)
- **Professional PDF Export** — Financial reports with cover page, table of contents, and charts
- **Excel & JSON Export** — Full data portability
- **Profiles** — Up to 3 separate profiles, each with its own encrypted database

## Security & Privacy

| | Tenivel |
|---|---|
| Data storage | 100% local on your device |
| Encryption | AES-256 (SQLCipher) |
| Server communication | License validation only — no financial data transmitted |
| Account | Optional, for license and device management |
| Tracking / Analytics | None in the app |
| Password protection | Yes, with local recovery key |
| Backups | Local, automatic + manual |
